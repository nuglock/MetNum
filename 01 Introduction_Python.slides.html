Operadores aritméticos¶

Como em uma calculadora, para executar operaes matemticas neccessitamos de operadores aritmticos. No Python, eles so:

  • $a+b$ -> Adio
  • $a-b$ -> Subtrao
  • $a/b$ -> Diviso
  • $a*b$ -> Multiplicao
  • $a$**$b$ -> Exponenciao
  • $a$%$b$ -> Diviso modular: Mostra o resto da diviso

Operadores relacionais¶

Para efetuar comparaes entre valores, precisamos do operadores relacionais. No Python, eles so:

  • $<$ -> Menor que
  • $>$ -> Maior que
  • $<=$ -> Menor ou igual a
  • $>=$ -> Maior ou igual a
  • $==$ -> Igual a
  • $!=$ -> Diferente de

Loops¶

Os loops so utilizados para realizar clculos repetitivos ou iterativos at que determinadas condies so satisfeitas. Os mais utilizados so o loop while e o for.

while¶

O while executa um bloco de comandos indetados se a enquanto a condio estabelecida seja verdadeira, ou seja, aps cada execuo do bloco de comandos, antes de se iniciar novamente a execuo, a condio novamente avaliada, e se verdadeira, repete-se novamente o ciclo at que a condio se torne falsa, veja o exemplo:

In [11]:
a_min = 0.3
a_lista = [] # Cria uma lista vazia
n = 1
while a > a_min:
    a = 1.0/n
    a_lista.append(a) # Adiciona um elemento  lista
    n = n + 1
print(a_lista)

[1.0, 0.5, 0.3333333333333333, 0.25]

Abrindo e Fechando Arquivos¶

Antes de acessar um arquivo armazenado, necessario criar um objeto de arquivo (file object):

file_object = open(nome_arquivo, ao)

Aqui voc cria o objeto de arquivo com nome igual a nome_arquivo, por ex. 'dados.txt' e define qual ao voc operar no arquivo. As aes podem ser:

  • r Ler um arquivo existente.

  • w Escrever em um arquivo. Se ele j no existir, ele criado.

  • a Acrescentar no fim de um arquivo.

  • r+ Ler e Escrever de um arquivo existente.

  • w+ Mesmo que r+, mas o arquivo criado se ele no existir.

  • a+ Mesmo que w+, mas os dados so acrescentados no final do arquivo.

Quando voc terminar de manipular o arquivo, deve-se fech-lo com

file_object.close()

Um outra forma de ler os dados de um arquivo pode ser vista no exemplo abaixo. O arquivo a ser lido "dados.txt' que contm dados de intensidade solar no formato (ano ms dia intensidade):

Funções e Módulos¶

Uma funo uma parte do programa que pode ser executada vrias vezes para retornar um valor. Por exemplo, a funo seno retorna o valor do seno de um ngulo fornecido, tambm chamado de argumento da funo.

Para definir uma funo, utilizamos o comando def:

def func_name(param1, param2,. . .):
    Operaes executadas
    return valor_retornado

O exemplo a seguir define uma funo que pode ser usada para defivar numericamente outra funo em um ponto x:

In [54]:
# Funo que deriva uma funo fornecida por diferenas finitas
def derivadas(f,x,h=0.0001):          # Define a funo derivadas e h tem um valor default ou padro
    df =(f(x+h) - f(x-h))/(2.0*h)          # Derivada primeira
    ddf =(f(x+h) - 2.0*f(x) + f(x-h))/h**2 # Derivada segunda
    return df,ddf                          # Retorna os valores das derivadas em x

Podemos, por exemplo, usar a funo derivadas para nos retornar as derivadas da funo seno no ponto $x=\pi/2$

Vetorizando algoritmos¶

Algumas vezes as propriedades inerentes s funes do mdulo mumpy podem ser utilizadas para substituir o uso de loops for ou while, conhecido como processo de vetorizao. Por exemplo, a expresso:

$ s=\sum_{i=0}^{100} \sqrt{\frac{i\pi}{100}}sin{\frac{i\pi}{100}}$

Para calcaular esse somatrio, poderamos utilizar um loop for como no exemplo a seguir, sem vetorizao:

In [47]:
from math import sqrt,sin,pi
x = 0.0; s = 0.0
for i in range(101):
    s = s + sqrt(x)*sin(x)
    x = x + 0.01*pi
print(s)

77.51389798916522

Ou com vetorizao:

In [5]:
from numpy import sqrt,sin,arange
from math import pi
x = arange(0.0, 1.001*pi, 0.01*pi)
print(sum(sqrt(x)*sin(x)))

77.5138979891651

Observe que o primeiro algoritmo importa as funes do mdulo math que no "conhecem" o objeto array. O segundo algoritmo importa as funes do mdulo numpy. A diferea entre eles que o segundo executa muito rpido mas utiliza mais memria.
